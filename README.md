<br />

「みんクイ ～みんなでクイズ～」は，参加プレイヤー全員がその場でクイズを作成し，リアルタイムに解き合う対戦ゲームです．

もともとは「研究室の飲み会とかのレクリエーションで，全員が主役になれるツールが欲しい」という友人のアイデアをきっかけに，私自身の手で形にしたものです
（おおまかなゲームルールも彼の発案です）．

既存のクイズアプリのような「用意された問題を解く」受動的な体験ではなく，自分たちの身内ネタをその場でコンテンツ化して盛り上がるライブ体験を設計しています．

_プレイは[こちら](https://minna-de-quiz.verce.app/)_

## 動画
